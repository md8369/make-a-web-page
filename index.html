Intro to Programming

UNDERSTANDING THE INTERNET

The internet is a group of computers sending and receiving messages. When a person types a web page it travels across the internet to servers. The servers look up the information requested and sends it back to the browser.

HTML

Hypertext Markup Language is the heart of the web. It contains text content that you can see and markup is how it looks.

TAGS

HTML documents are created using tags. Tags have an opening and a closing depending on which one you are using. Examples of tags are bold,emphasis, anchor and linkable tags to display images or website.

Creating a Structured Document

DEVELOPER TOOLS

HTML are either inline or block. Block elements form an invisible box around the content. HTMLand CSS are both languages. HTML controls the structure of a webpage. CSS controls the style of a page. When programmers talk about the DOM (Document Object Model) they are talking about the tree-like structure of a page. HTML is the standard markup language used to create web pages. CSS is the style sheet language used for describing the look and format of document written in a markup language .

TREE-LIKE STRUCTURE

The tree-like structure of HTML comes from the fact that HTML elements and have other elements inside of them. You can draw the relationship like a family tree. In a browser it shows up as a series of nested boxes. When reading an HTML document as text, you see a wave of changing indentations going up and down the left side of the document.

TEXT EDITORS FOR PROGRAMMERS

When writing a code, programmers use special text editors like sublime text. They make life easier. For example, some text editors will automatically adding closing tag.

Adding Color and Style to Code

INHERITANCE

Inheritance is the feature in CSS. It relies on ancestor-descendant relationship to operate. Inheritance is the mechanism by which properties are applied not only to speicified elements but also its dscendants. Properties that can be inherited are color,font,letter-spacing,line-height,list-style,text-align,text-indent, text-tranform,visibility,whitespace and word spacing. Properties that cannot be inherited are background,border, height,width ,outline, padding, position and text decoration just to name a few.

Code comments are lines of text that the computer ignores. They are just notes the programmer can use for other people to see.
